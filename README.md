# Projetos CS50

Exercícios fundamentais desenvolvidos durante o curso de Ciências da Computação (CC50).

## Sobre
Este repositório contém as soluções dos desafios da primeira semana, focados em lógica de programação em C.

## Conteúdo
- **Hello:** Introdução à sintaxe básica.
- **Mario:** Lógica de loops aninhados.
- **Cash:** Algoritmos gulosos (*greedy algorithms*).
- **Population:** Estruturas de repetição `while`.
